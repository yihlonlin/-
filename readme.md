課程實作專題說明
==
自然語言處理屬資訊進階課程，本專題實作目的是設計一套理論與實作整合、軟體與硬體整合及人工智慧與醫學語音助理應用整合。該專題主要區分七個專題的學習路徑，以從專題的困難度面向來看，可區分基礎、進階與挑戰。以學習模式看來，可區分個人表現與團隊合作分組方式。學習過程中，以手把手的教學與引導式方式教學，期末學期成果需要進行成果展演。該課程的價值是讓學生可以瞭解自然語言處理與醫學助理專題實作的知識與實踐。實作專題名稱如下：
- 實作1. 使用中文斷詞搜尋義大醫師命名實體實作-(個人)(基礎)
- 實作2. 使用網路爬蟲顯示醫師門診時間專題實作-(個人)(基礎)
- 實作3. 使用 LineBot 訊息聊天機器人查詢醫師門診時間實作-(個人)(基礎)
- 實作4. 使用 LineBot 訊息聊天機器人量表檢測實作-(分組)(進階)
- 實作5. 使用 Google Voice Kit 回應門診時間語音助理實作-(分組)(進階)
- 實作6. 使用深度學習實作骨鬆病例分類-(分組)(挑戰)
- 實作7. 使用深度學習實作 AI Cup 生醫關聯擷取-(分組)(挑戰)
